# CapitalOne
This is a private repository.


##### Environment
```
OS X    10.11.6
NodeJS  6.11.4
npm     3.10.10
```

##### Build
1. Be sure to `nvm use` to set correct NodeJS version.
2. `npm install` to download dependencies.
